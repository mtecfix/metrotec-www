# MetroTec Website

Professional IT services website for Metro Detroit businesses within 25-mile radius.

## Main Pages
- `index.html` - Homepage with hero, services overview, and contact
- `services.html` - Detailed services page
- `industries.html` - Industries served page  
- `about.html` - Company information and contact details

## Service Pages
- `managed-it.html` - Managed IT Services
- `cybersecurity.html` - Cybersecurity Services
- `cloud-services.html` - Cloud Services & Migration
- `voip-phone.html` - VoIP Phone Systems

## Industry Pages
- `healthcare.html` - Healthcare IT Solutions
- `manufacturing.html` - Manufacturing IT Solutions
- `professional-services.html` - Professional Services IT
- `retail.html` - Retail IT Solutions

## Core Files
- `styles.css` - Main stylesheet with 60-30-10 color scheme
- `script.js` - JavaScript for navigation and form interactions

## Features
- Responsive design for mobile and desktop
- 60-30-10 color rule implementation
- SEO-optimized content for Metro Detroit area
- Contact forms and call-to-action buttons
- Vendor partnership links

## Service Area
Metro Detroit within 25-mile radius including Detroit, Dearborn, Southfield, Troy, Warren, Sterling Heights, Livonia, Westland, Farmington Hills, and surrounding communities.
